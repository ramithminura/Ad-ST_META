Phase 0 — Do This First
Install the Meta Pixel
Everything depends on this. Retargeting, BOFU, lookalike audiences — none of it works without the pixel firing on your site.
0.1
Create a Meta Pixel in Events Manager
+
  1. Go to business.facebook.com → open the menu → Events Manager
  2. Click Connect Data Sources → select Web → click Get Started
  3. Name your pixel: Maera Jewellery Pixel → enter your website URL → click Continue
  4. Select Shopify as your partner → click Next
Meta will give you a Pixel ID — copy it. You'll paste it into Shopify in the next step.
0.2
Connect Pixel to Shopify
+
  1. In Shopify Admin → go to Sales Channels → Facebook & Instagram
  2. If not installed: click Add sales channel → Facebook & Instagram
  3. Click Settings → Data Sharing → set to Maximum
  4. Under Meta Pixel, paste your Pixel ID → click Confirm
  5. Enable Customer events — this powers Conversions API (server-side tracking)
Do NOT skip setting Data Sharing to Maximum — this enables server-side Conversions API which dramatically improves tracking accuracy, especially on iOS devices.
0.3
Verify pixel is firing correctly
+
  1. Install the Chrome extension Meta Pixel Helper
  2. Visit your website — the extension icon should turn green and show PageView firing
  3. In Events Manager → Test Events → enter your website URL → click Open Website
  4. Browse your site — confirm PageView, ViewContent, and any collection events appear in the test panel
Give the pixel 24 hours to start registering traffic data before launching campaigns. You can set up campaigns now and launch tomorrow.
Phase 1 — Before You Build Campaigns
Account Foundations
Set up the building blocks every campaign will pull from. Do this once — everything else references it.
1.1
Set your ad account currency, time zone & spending limit
+
  1. Meta Ads Manager → Settings → Ad Account Settings
  2. Currency: CAD (Canadian Dollar — matches your budget)
  3. Time zone: Set to your operating time zone (Colombo = UTC+5:30)
  4. Set an Account Spending Limit of CAD $870 — this is a hard ceiling so you never overspend

Phase 2 — Campaign A
Awareness Campaign — TOFU
This campaign does one thing: get SURYA in front of cold audiences elegantly. No selling. No CTA that says "Buy". Pure signal.
2.1
Create Campaign A — settings
+
  1. Ads Manager → Create → select Awareness objective
  2. Name: SURYA — Campaign A — Awareness TOFU
  3. Buying type: Auction
  4. Campaign budget optimisation: OFF — you'll control budget at ad set level
  5. Campaign spending limit: leave blank (controlled at ad set level)
  6. Advantage campaign budget: OFF
Keep Advantage+ OFF for Weeks 1–2. You need manual control to kill underperformers fast. Only turn it on for proven performers in Week 3.

Phase 3 — Campaign B
Sales Campaign — MOFU + BOFU
This campaign converts warmer audiences. Education first (MOFU), then scarcity (BOFU). Run BOFU to retargeting audiences only — never cold.
3.1
Create Campaign B — settings
+
  1. Create → objective: Leads (use this instead of Sales — you're converting to DM enquiry, not checkout)
  2. Name: SURYA — Campaign B — MOFU + BOFU
  3. Buying type: Auction
  4. Campaign Budget Optimisation: OFF
Use Leads objective, not Sales — your conversion is a DM or contact form enquiry, not a tracked purchase. This tells Meta to find people likely to message or click through, not checkout.

4.3
When to launch Google Ads
+

Launch Google on Day 10–11 (mid-Week 2). By then:

  1. Your Meta TOFU campaigns have been running ~10 days — people are searching for you
  2. Your pixel has enough data for Google to create a Customer Match audience from site visitors
  3. You'll know which geographic markets are actually responding (use that to inform Google geo targeting)
Google Search captures people actively searching "Ceylon sapphire ring" or "Maera Jewellery" — these are your highest-intent buyers. Budget: CAD $140 Search + CAD $70 Display Retargeting.
