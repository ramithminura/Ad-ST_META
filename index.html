Phase 0 — Do This First
Install the Meta Pixel
Everything depends on this. Retargeting, BOFU, lookalike audiences — none of it works without the pixel firing on your site.
0.1
Create a Meta Pixel in Events Manager
+
  1. Go to business.facebook.com → open the menu → Events Manager
  2. Click Connect Data Sources → select Web → click Get Started
  3. Name your pixel: Maera Jewellery Pixel → enter your website URL → click Continue
  4. Select Shopify as your partner → click Next
Meta will give you a Pixel ID — copy it. You'll paste it into Shopify in the next step.
0.2
Connect Pixel to Shopify
+
  1. In Shopify Admin → go to Sales Channels → Facebook & Instagram
  2. If not installed: click Add sales channel → Facebook & Instagram
  3. Click Settings → Data Sharing → set to Maximum
  4. Under Meta Pixel, paste your Pixel ID → click Confirm
  5. Enable Customer events — this powers Conversions API (server-side tracking)
Do NOT skip setting Data Sharing to Maximum — this enables server-side Conversions API which dramatically improves tracking accuracy, especially on iOS devices.
0.3
Verify pixel is firing correctly
+
  1. Install the Chrome extension Meta Pixel Helper
  2. Visit your website — the extension icon should turn green and show PageView firing
  3. In Events Manager → Test Events → enter your website URL → click Open Website
  4. Browse your site — confirm PageView, ViewContent, and any collection events appear in the test panel
Give the pixel 24 hours to start registering traffic data before launching campaigns. You can set up campaigns now and launch tomorrow.

Phase 0 — Do This First
Create Your Google Ads Account
Launch this on Day 10–11, once Meta has been running for 10 days. By then people will be searching for you.
0.1
Create Google Ads account
+
  1. Go to ads.google.com → click Start now
  2. Sign in with your Google account (use the same account as your Google Analytics if you have it)
  3. When asked your main advertising goal → select "Get more website sales or sign-ups"
  4. Google will try to push you into Smart campaigns — click "Switch to Expert Mode" at the bottom
  5. Select "Create an account without a campaign" — you will set campaigns up manually
  6. Set billing country, time zone (Sri Lanka UTC+5:30), currency: CAD
  7. Enter billing details and set a monthly spending limit of CAD $220
Do NOT let Google auto-create a Smart Campaign. Always click "Switch to Expert Mode" — Smart campaigns give you no control over keywords, match types, or bids.
0.2
Install Google Tag (conversion tracking)
+
  1. In Google Ads → Tools → Measurement → Conversions → click + New conversion action
  2. Select Website → enter your website URL → click Scan
  3. Create a conversion action: name it Enquiry / Contact Form · Category: Submit lead form · Value: Don't assign
  4. Create a second: name it Page View — Collection · Category: Page view
  5. Google will give you a tag snippet — in Shopify Admin → Online Store → Themes → Edit code → theme.liquid
  6. Paste the Google Tag snippet just before the </head> closing tag → Save
  7. Use Google Tag Assistant Chrome extension to verify it's firing on your site
When you move to a custom website, this tag moves with you — just paste it into the new site's <head>. No need to recreate conversion actions.
0.3
Link Google Ads to Google Analytics (if you have it)
+
  1. Google Ads → Tools → Linked accounts → Google Analytics
  2. Select your GA4 property → click Link
  3. Enable Import Analytics conversions — this imports goals from GA into Google Ads
If you don't have Google Analytics yet, skip this step. Set it up later — it won't block your campaigns from running.

  2. Look for high-performing terms you didn't think of → right-click → Add as keyword (add as exact match)
  3. Look for French queries appearing → if strong, create a separate French ad group with dedicated French-language ad copy
  4. Export the report weekly → build your negative keyword list progressively
The Search Terms report is where Google Ads gets better over time. The more negative keywords you add, the less money you waste. Treat it like a weekly ritual — 10 minutes every 4 days.

How Meta + Google work together
+

The two platforms amplify each other — this is the echo chamber effect in action:

  1. Meta TOFU creates awareness → person sees SURYA on Instagram for the first time
  2. Person searches "Ceylon sapphire ring" on Google after seeing the Instagram content
  3. Google Search ad intercepts that search → they click through to your site
  4. Google Display retargeting follows them across other websites → reinforces SURYA
  5. Meta BOFU static retargets them on Instagram → "You have already seen it."
  6. DM enquiry → conversion
When reviewing performance, look at assisted conversions in Google Analytics — not just last-click. Meta's contribution often shows up as assisted, not direct, because the final click comes through Google.
